# Cadastro de Produtos

Projeto Flutter + Spring Boot para **cadastro, listagem, edição e exclusão de produtos**, usando **H2 Database**.  
Permite validar campos e mostrar mensagens de sucesso/erro no frontend.

---

## Estrutura do Projeto

- `backend/` → Spring Boot
- `flutter/` → Flutter App
- `README.md` → Este arquivo

---

## Pré-requisitos

- Java 17+  
- Maven ou Gradle  
- Flutter 3.7+  
- Navegador Chrome (para Flutter Web) ou emuladores Android/iOS  
- IDE de sua preferência (VSCode, IntelliJ, Android Studio)  

---

## Flutter Web

- flutter run -d chrome --web-port 4200
- flutter run -d edge --web-port 4200
- flutter run -d firefox --web-port 4200
